# 🎮 Tic Tac Toe Game

An interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project demonstrates basic game logic, DOM manipulation, and user interaction.

## 🚀 Features
- Two-player mode (Player X vs Player O)
- Real-time game updates
- Win detection using patterns
- Draw condition handling
- Restart game functionality
- Clean and responsive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 How to Run
1. Download or clone this repository
2. Open `tictactoe.html` in any web browser
